# JFG-z
Jet Force Gemini tools.

### Usage

File locations:
```
Bizhawk/
  Lua/
    JetForceGemini.lua
    JetForceGemini/
		core.lua
		data/
			common.lua
			NJFE0.lua
			versions.lua
		utils/
			hex.lua
			memory.lua
```

Have a ROM loaded then run the `JetForceGemini.lua` script.
